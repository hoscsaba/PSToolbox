Connects SCP front to reservoir

Parameters
t_target[in] target time value
*r1[in] pointer to reservoir
*p1[in] pointer to pipe
rho[in] density, kg/m3
a[in] dsonic velocity, m/s
inletpressure drop [in] bool, if true, inlet pressure drop issumed, i.e. pr=p+rho/2*v^2
p[out] pressure at the first node of the pipe, result of computations
