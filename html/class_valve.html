void Valve::SetAeffCoeffs ( const double _a1,
const double _a2,
const double _a3,
const double _AeffMax )

Set the coefficients for the Aeff(x) function The approximating function will be $A_{eff}=min(A_{eff,max},1+a_1 y+a_2 y^2+a_3 y^3)$, where $y$ is the relative lift: $y=x/(D/4)$

Parameters
_a1[in] coeff. of y
_a2[in] coeff. of y^2
_a3[in] coeff. of y^3
_Aeffmax[in] fixed value for y>1

Set coefficients for the Cd(x) function. The approximating function will be $C_d=b_0+b_1 y+b_2 y^2+b_3 y^3)$, where x is the relative lift: $y=x/(D/4)$

Parameters
_b0[in] constant term (set this for fixed Cd value)
_b1[in] coeff. of y
_b2[in] coeff. of y^2
_b3[in] coeff. of y^3
